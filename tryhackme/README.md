# TryHackMe Learning Log

This folder contains high-level documentation of hands-on labs completed on TryHackMe.

⚠️ Notes in this folder do not include walkthroughs, commands, flags, or step-by-step exploit instructions.
All content focuses on concepts learned, attacker and defender perspectives, and security takeaways.

## Purpose
- Reinforce cybersecurity fundamentals through guided labs
- Build an attacker mindset in a safe, ethical environment
- Translate hands-on experience into interview-ready explanations

## How This Folder Is Used
Each TryHackMe section or room is documented using:
- A brief description of the objective
- Key concepts learned
- High-level attacker perspective
- Defensive considerations and mitigations
- An interview-ready summary

## Disclaimer
All exercises referenced here were completed in a controlled lab environment for educational purposes only.
